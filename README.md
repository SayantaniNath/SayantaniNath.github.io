### [SF Take-Home & Savings Calculator](https://sayantaninath.github.io/SF_Salary_Calculator.html)
Interactive salary calculator for San Francisco. Enter one or two salaries to see:
- Federal + California state tax breakdown (2025 brackets)
- FICA (Social Security + Medicare) and CA SDI
- 401(k) pre-tax contribution impact
- Monthly cash flow: take-home → expenses → liquid savings → total savings
- What-if salary scenarios
- Editable SF cost-of-living expense defaults (rent, groceries, transport, etc.)
